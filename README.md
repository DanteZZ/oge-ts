Open-source lite game framework inspired by GameMaker and Unity. Written in TypeScript.

## Installing

With npm or yarn:

```bash
npm i oge-ts
```

```bash
yarn add oge-ts
```

With browser:

```html
<script src="https://raw.githubusercontent.com/DanteZZ/oge-ts/main/dist/oge.min.js"></script>
```
